■개요
 - 프로젝트 이름: 키오스크 만들기
 - 프로젝트 기간: 2025.03.10 - 2025.03.13
 - 개발 언어: java 

■설명
 : 비대면 주문으로 일부 면에 특정 메뉴가 출력되어 원하는 메뉴를 입력받아 주문 가능하게 해줍니다.

■사용 방식
 1) 프로그램 실행
 2) 메인 메뉴 카테고리 출력
 4) 출력 메세지에 맞게 메뉴 카테고리 번호 입력
 5) 해당 카테고리의 상세 메뉴 출력
 6) 출력 메세지에 맞게 메뉴 카테고리 번호 입력
 7) 선택한 메뉴명과 가격, 메뉴 설명 표시
 8) 장바구니 추가를 원하면 1번 확인, 그렇지 않으면 2번 취소 입력
 8-1) 2번 취소를 입력하면 다시 메인 메뉴 화면으로 이동
 8-2) 1번 확인을 입력하면 장바구니에 메뉴 추가
 8-3) 추가하면 추가되었다는 메세지 출력
 9) 다시 메인메뉴 출력 (장바구니 메뉴가 추가될경우, 장바구니 메뉴 카테고리도 출력)
 10) 4번 주문 카테고리 입력
 11) 주문 확인 메세지와 함께 장바구니 메뉴 출력
 11-1) 현재 장바구니에 담긴 메뉴 금액도 출력
 12) 주문을 원하면 1번 주문, 그렇지 않으면 2번 메뉴판 입력
 12-1) 2번 메뉴판을 입력하면 다시 메뉴 카테고리로 이동
 12-2) 1번 주문을 입력하면 주문완료, 총 금액 메세지 출력
 13) "이용해주셔서 감사합니다." 메세지 출력
 14) 프로그램 종료

■파일 설명
 - Bascket : 장바구니 관련된 메뉴 및 메세지 출력 기능 관리 클래스
 - InputValidator : 입력 예외처리 클래스
 - Kiosk : 키오스크 입출력 기능 클래스
 - Main : Main 함수 클래스
 - Menu : 메뉴 카테고리 및 메뉴 생성, 추가 기능 관리 클래스
 - MenuItem : 상세 메뉴 클래스
